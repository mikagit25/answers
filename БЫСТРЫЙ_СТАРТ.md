
## 🐛 Возможные проблемы и решения

### Проблема: Docker не запускается
**Решение**: Перезагрузите компьютер и запустите Docker Desktop

### Проблема: "Connection refused" при подключении к БД
**Решение**: Подождите 10 секунд после запуска `docker-compose up -d`

### Проблема: Нехватка памяти
**Решение**: 
- Закройте браузер и другие приложения
- Уменьшите лимиты в Docker Desktop (Settings → Resources → Memory: 4GB)

### Проблема: OpenRouter не работает
**Решение**: 
- Проверьте ключ в `backend/.env`
- Убедитесь что интернет работает
- Проверьте статус: https://openrouter.ai/status

```bash
curl http://localhost:8000/api/v1/providers/status | python3 -m json.tool
```

### Посмотреть логи backend:
В терминале с backend будут видны сообщения:
- `🤖 Using LLM provider: openrouter` - какой провайдер используется
- `⚠️ Rate limit hit...` - переключение на другой провайдер
- `❌ Error with...` - ошибки провайдера
