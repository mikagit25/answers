# 🌍 Мультиязычная поддержка - 12 языков мира

## ✅ Реализовано

Проект теперь поддерживает **12 основных языков планеты** с автоматическим определением языка браузера и возможностью переключения.

## 🎯 Как использовать

### 1. Переключатель языков

В правом верхнем углу интерфейса находится кнопка с текущим языком:
```
🇬🇧 English ▼
```

При нажатии открывается выпадающий список со всеми 12 языками.

### 2. Автоматическое определение

При первом посещении система автоматически определяет язык вашего браузера и переключает интерфейс.

### 3. Сохранение предпочтений

Выбранный язык сохраняется в localStorage и восстанавливается при следующем посещении.

## 🔧 Технические детали

### Backend API

#### Получить список языков:
```bash
GET /api/v1/languages
```

**Response:**
```json
{
  "languages": [
    {
      "code": "en",
      "name": "English",
      "native_name": "English",
      "flag": "🇬🇧",
      "direction": "ltr"
    },
    // ... еще 11 языков
  ]
}
```

#### Задать вопрос с указанием языка:
```bash
POST /api/v1/ask
Content-Type: application/json

{
  "question": "Как преодолеть страх?",
  "tradition_id": "stoicism",
  "language": "ru"  // Опционально
}
```

Если язык не указан, система определяет его из заголовка `Accept-Language`.

#### Сравнение традиций с языком:
```bash
POST /api/v1/compare
Content-Type: application/json

{
  "question": "What is the meaning of life?",
  "exclude": "stoicism",
  "language": "en"
}
```

## 💡 Особенности реализации

### 1. Автоматический перевод ответов

LLM получает инструкцию отвечать на языке вопроса:

**Для русского:**
```python
prompt = f"""Ты эксперт по {tradition_name}. 
Ответь на этот вопрос основываясь на философии {tradition_name}:
Вопрос: {request.question}
Дай краткий, полезный ответ на русском языке."""
```

**Для арабского:**
```python
prompt = f"""أنت خبير في {tradition['name']}.
أجب على هذا السؤال باللغة العربية..."""
```

**Для остальных:**
```python
prompt = f"""Answer in the same language as the question."""
```

### 2. Перевод названий традиций

Названия традиций также переводятся:
- Стоицизм → Stoicism → Estoicismo → 斯多葛主义 → الرواقية
- И т.д. для всех 12 языков

### 3. Полный перевод UI

Все элементы интерфейса переведены:
- Кнопки
- Placeholder'ы
- Сообщения об ошибках
- Метаданные
- Статусы загрузки

## 📝 Добавление новых языков

Чтобы добавить новый язык:

1. **Добавить в SUPPORTED_LANGUAGES:**
```python
"ko": {
    "name": "Korean",
    "native_name": "한국어",
    "flag": "🇰🇷",
    "direction": "ltr"
}
```

2. **Добавить переводы в TRANSLATIONS:**
```python
"ko": {
    "app_title": "답변 플랫폼",
    "app_subtitle": "8개의 철학적 및 종교적 전통에서의 지혜",
    # ... остальные ключи
}
```

3. **Обновить логику промптов:**
Добавить условие для корейского языка в `test_server.py`
