## 🔧 Технические детали

### Как работает система переводов:

**1. Централизованный объект translations:**
```javascript
const translations = {
    'en': {
        'tradition_stoicism': 'Stoicism',
        'get_answer': 'Get Answer',
        // ... все ключи
    },
    'ru': {
        'tradition_stoicism': 'Стоицизм',
        'get_answer': 'Получить ответ',
        // ... все ключи
    }
    // ... еще 10 языков
};
```

**2. Функция getTranslation():**
```javascript
function getTranslation(key) {
    if (translations[currentLang] && translations[currentLang][key]) {
        return translations[currentLang][key];
    }
    // Fallback to English
    if (translations['en'][key]) {
        return translations['en'][key];
    }
    return key; // Если ничего не найдено, вернуть ключ
}
```

**3. Автоматический перевод через data-translate:**
```html
<label data-translate="select_tradition">Select a Tradition:</label>
<button data-translate="get_answer">Get Answer</button>
```

**4. Обновление при смене языка:**
```javascript
function updateUITranslations() {
    // Найти все элементы с data-translate
    document.querySelectorAll('[data-translate]').forEach(element => {
        const key = element.getAttribute('data-translate');
        element.textContent = getTranslation(key);
    });
    
    // Обновить placeholders
    questionInput.placeholder = getTranslation('question_placeholder');
    
    // Обновить depth опции
    updateDepthOptions();
}
```
