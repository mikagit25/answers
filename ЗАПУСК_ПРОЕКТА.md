## 🐛 Решение проблем

### Проблема: Сервер не запускается
**Решение**: Проверьте что порт 8000 свободен
```bash
lsof -i :8000
```

### Проблема: Ошибка API ключа
**Решение**: Проверьте файл `backend/.env`
```bash
cat backend/.env | grep GROQ_API_KEY
```

### Проблема: Ответы на английском
**Решение**: Это нормально - модель отвечает на языке вопроса. Задавайте вопросы на русском.
