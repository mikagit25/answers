## 🚀 Как запустить полный проект

### Вариант 1: Без Docker (быстрый старт для тестирования)

Так как Docker устарел (версия 17.03), можно запустить backend без PostgreSQL:

```bash
cd "/Volumes/NEW 1/Answers/backend"
source venv/bin/activate

# Запустить backend с SQLite (для тестирования)
# Нужно создать простую версию main.py с SQLite
uvicorn main_simple:app --reload
```

### Вариант 2: Обновить Docker (рекомендуется для продакшена)

```bash
# Обновить Docker Desktop до последней версии
# Скачать: https://www.docker.com/products/docker-desktop/

# После обновления:
docker-compose up -d
cd backend && python ingest.py
uvicorn main:app --reload
```
