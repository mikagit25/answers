### 2. Backend (FastAPI + Python)

#### Core Components
- ✅ **Configuration Management** (`backend/core/config.py`)
  - All environment variables configured
  - RAG pipeline settings (chunk size, overlap, top-k retrieval)
  
- ✅ **Database Layer** (`backend/db/`)
  - SQLAlchemy models: Tradition, Source, Chunk, UserQuery
  - Async PostgreSQL connection with pgvector support
  - Vector similarity search indexes
  
- ✅ **RAG Pipeline** (`backend/rag/`)
  - **prompts.py**: System prompts for all 8 traditions (Stoicism, Christianity, Islam, Buddhism, Judaism, Hinduism, Taoism, Humanism)
  - **pipeline.py**: Complete RAG workflow (retrieve → generate → validate)
  - **validator.py**: Post-validation of sources (checks for author mentions, work titles, quote presence)
  
- ✅ **API Routes** (`backend/api/`)
  - `POST /api/v1/ask` - Question answering endpoint
  - `GET /api/v1/compare` - Tradition comparison endpoint
  - `GET /health` - Health check
  - Pydantic schemas for request/response validation
  
- ✅ **Security** (`backend/core/security.py`)
  - Rate limiting middleware (60 requests/minute)
  - Input validation and sanitization
  - XSS prevention patterns

- ✅ **Data Ingestion** (`backend/ingest.py`)
  - Imports JSON knowledge base into PostgreSQL
  - Generates embeddings using sentence-transformers
  - Creates vector chunks with overlap

## 🚀 How to Run

### Quick Start (Automated)
```bash
./setup.sh
```

### Manual Start

**1. Start Infrastructure:**
```bash
docker-compose up -d
```

**2. Import Data:**
```bash
cd backend
python ingest.py
```

**3. Run Backend:**
```bash
cd backend
uvicorn main:app --reload --port 8000
```

**4. Run Frontend:**
```bash
cd frontend
npm run dev
```

**5. Open Browser:**
Visit [http://localhost:3000](http://localhost:3000)

---

## 🔧 Configuration Required

Before running, edit `backend/.env`:

```bash
DATABASE_URL=postgresql://user:password@localhost:5432/answers_db
REDIS_URL=redis://localhost:6379
QWEN_API_KEY=your_actual_qwen_api_key_here
OPENROUTER_API_KEY=optional_if_using_openrouter
```

**Get Qwen API Key:**
- Direct: https://help.aliyun.com/zh/model-studio/developer-reference/get-api-key
- Or use OpenRouter: https://openrouter.ai/
